# Effects Library for BE1st Platform

## Quick Reference

| Effect | Use Case | Industries |
|--------|----------|------------|
| ScrollReveal | Content sections, cards | ALL |
| AnimatedCounter | Statistics, metrics | ALL |
| TiltCard | Product cards, team members | tech, ecommerce |
| ParallaxSection | Hero backgrounds | luxury, realty |
| TypeWriter | Headlines, taglines | tech, saas |
| MagneticButton | CTAs, buttons | tech, creative |
| SoftBounce | Icons, small elements | handmade, playful |
| GlowEffect | Premium highlights | luxury, tech |
| StaggeredList | Lists, grid items | ALL |
| FloatingElements | Background decoration | creative |

## Usage

import { ScrollReveal, AnimatedCounter } from '../effects';

<ScrollReveal>
  <YourContent />
</ScrollReveal>

<AnimatedCounter end={5000} suffix="+" />
